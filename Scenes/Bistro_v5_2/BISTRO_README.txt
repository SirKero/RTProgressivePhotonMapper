1. Download the Bistro scene here: https://developer.nvidia.com/orca/amazon-lumberyard-bistro

2. Unpack the scene into the Repo/Scenes/Bistro_v5_2 folder so that it looks like this:
	- Texture
	- BistroExterior.fbx
	- ...
	- RTPM_BistroInterior_Wine_PerformanceTestPosition.pyscene

3. Both pyscenes can now be loaded in by the Mogwai renderer.
